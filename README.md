# PROCONSVATE (React + TypeScript)

Este projeto foi migrado para **React com TypeScript (TSX)**, mantendo o mesmo conteúdo visual das páginas originais:

- Início
- Notícias
- Quiz
- Sobre / PROCONSVATE

## Executar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Observações

- Estilos em **CSS puro** (`css/global.css`), sem Tailwind.
- O conteúdo da antiga `index.html` foi migrado para `src/App.tsx`.
